Peanotes 1.3.x

Peanotes (PNotes от PHP Notes) - мой учебный проект. Более того - мой первый проект на PHP.

По изначальной задумке он должен был уметь сохранять, выводить на экран, редактировать и удалять заметки. Но позже мне пришла в голову идея развития его функционала. В будущем тут должны появиться:

 * Адаптивный дизайн для маленьких экранов

 * Инсталятор для удобной установки на сервер.

 * Поиск по заметкам.

 * Пагинатор

 * Сортировка заметок по дате создания или редактирования

 * Пакетное удаление заметок

 * Регистрация и авторизация пользователей

 * Личная переписка

 * Таксономия

 * Корзина

К сожалению у меня катастрофически мало времени, поэтому развивается приложение не очень быстро.
